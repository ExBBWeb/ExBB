Структура базы данных.

Таблица #_extensions
section - определяет, где действует расширение.
	1 - форум,
	2 - админ-панель
type - тип расширения
	0 - неопределённый тип
	1 - модуль
	2 - плагин
	3 - шаблон
	4 - язык
	5 - виджет
name - название папки расширения на сервере
title - название расширения
description - описание расширения
author - разработчик расширения
author_url - сайт разработчика
version - версия расширения (подробнее о версиях в разделе "Обновление движка, система версий")
